## function
sign function
$$
y = sgn x = \left\{\begin{matrix}
 -1, & x < 0 \\ 
 0,  & x = 0 \\
 1,  & x > 0
\end{matrix}\right.
$$ 
--------
INT function
$$
y = [x]\\
The Basic Inequality:
x-1<[x]\le x
$$
